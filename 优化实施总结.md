#### 核心代码

```java
// TokenService.java - Token撤销
public void revokeAllUserTokens(String tenantId, Long userId) {
    // 清除用户缓存
    tokenCacheService.evictUser(tenantId, userId);
    tokenCacheService.evictTenant(tenantId);
    
    // 在Redis中记录撤销时间戳
    String revocationKey = "token:revocation:" + tenantId + ":" + userId;
    redisTemplate.opsForValue().set(revocationKey, 
            String.valueOf(System.currentTimeMillis()), 
            Duration.ofHours(24));
}

// TokenService.java - Token验证时检查撤销
public boolean isTokenRevokedForUser(String tenantId, Long userId, long tokenIssuedAt) {
    String revocationKey = "token:revocation:" + tenantId + ":" + userId;
    String revocationTime = redisTemplate.opsForValue().get(revocationKey);
    
    if (revocationTime != null) {
        long revokedAt = Long.parseLong(revocationTime);
        return tokenIssuedAt < revokedAt;  // Token在撤销前签发，视为已撤销
    }
    return false;
}

// AuthController.java - 密码修改时调用
@PostMapping("/change-password")
public ResponseEntity<ApiResponse<BaseAuthResponse>> changePassword(@Validated @RequestBody AuthRequests.ChangePassword req) {
    User user = passwordAuthService.changePassword(req);
    
    // 撤销所有现有token
    tokenService.revokeAllUserTokens(req.getTenantId(), user.getId());
    
    return ResponseEntity.ok(ApiResponse.success(buildBaseAuthResponse(user, req.getTenantId())));
}
```

#### 安全提升
- ✅ 密码修改后立即撤销所有token
- ✅ 强制所有设备重新登录
- ✅ 防止密码泄露后的持续访问
- ✅ 撤销信息存储24小时（超过最大token TTL）

#### 新增错误码
- `TOKEN_REVOKED` (400203): Token has been revoked due to security action

### 2.3 租户缓存TTL独立调优（超高流量）

#### 实现文件
- `src/main/java/com/mercury/auth/config/CacheConfig.java` (重构)
- `src/main/resources/application-prod.yml` (修改)

#### 核心代码

```java
// CacheConfig.java - 独立缓存配置
@Configuration
@EnableCaching
public class CacheConfig {
    
    @Value("${security.token-cache.expire-after-write-seconds:600}")
    private long tokenExpireAfterWriteSeconds;  // 10分钟
    
    @Value("${security.tenant-cache.expire-after-write-seconds:1800}")
    private long tenantExpireAfterWriteSeconds;  // 30分钟
    
    @Bean
    public CacheManager cacheManager() {
        // Token相关缓存：短TTL（安全优先）
        Cache tokenCache = buildCache("tokenCache", maxCacheSize, tokenExpireAfterWriteSeconds);
        Cache userStatusCache = buildCache("userStatusCache", maxCacheSize, tokenExpireAfterWriteSeconds);
        
        // 租户缓存：长TTL（性能优先）
        Cache tenantStatusCache = buildCache("tenantStatusCache", tenantCacheMaxSize, tenantExpireAfterWriteSeconds);
        
        return cacheManager;
    }
}
```

#### 配置示例

```yaml
# application-prod.yml
security:
  token-cache:
    max-size: 10000
    expire-after-write-seconds: 600  # 10分钟
  tenant-cache:
    max-size: 1000
    expire-after-write-seconds: 1800  # 30分钟
```

#### 环境变量

```bash
# Token缓存
TOKEN_CACHE_MAX_SIZE=10000
TOKEN_CACHE_EXPIRE_SECONDS=600

# 租户缓存（新增）
TENANT_CACHE_MAX_SIZE=1000
TENANT_CACHE_EXPIRE_SECONDS=1800
```

#### 性能提升
- ✅ 减少租户验证数据库查询
- ✅ 租户状态变更最多30分钟生效（可配置）
- ✅ 超高流量场景性能优化
- ✅ 不同缓存类型独立调优
